# Acme Product Overview

Acme is a workspace tool for distributed teams. Features:

- Real-time collaborative documents
- Threaded discussions
- Native video calls (up to 25 participants)
- Searchable knowledge base
- API and webhooks
- Single sign-on (SSO) via Okta, Google, Microsoft

## Supported platforms

Web (Chrome, Safari, Firefox, Edge), macOS desktop, Windows desktop, iOS, Android. Linux is in beta.

## Data residency

Workspaces created on the EU plan store data in Frankfurt (eu-central-1). All other workspaces store in us-east-1. Cross-region migration is available on the Enterprise plan.
