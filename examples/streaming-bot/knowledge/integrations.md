# Integrations

Acme integrates natively with:

- **Slack** — bidirectional message sync; @acme commands inside Slack
- **GitHub** — link PRs and issues to documents; comment sync
- **Jira** — issue creation and linking
- **Linear** — issue creation and linking
- **Google Calendar / Outlook** — meeting notes attached to events
- **Zapier** — 5,000+ apps via Zapier triggers
- **REST API** — public, OAuth 2.0, rate-limited to 600 requests/minute per workspace
- **Webhooks** — outbound, signed with HMAC-SHA256

Integrations are available on Pro and Enterprise plans. Free plan users can use the API in read-only mode.
