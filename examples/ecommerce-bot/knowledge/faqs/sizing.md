# Sizing FAQ

- All jackets and tops use US standard sizing.
- Pima T-shirts (Echo) have a relaxed cut — size down for slim fit.
- Horizon Sneakers run true-to-size. Half-size up for thicker socks.
- Atlas Down Jacket runs slightly slim through the chest if layering over a thick sweater.
- We do not currently offer half sizes outside footwear.

If unsure between two sizes, order both — returns are free within 30 days.
